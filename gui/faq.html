HTML Conversion

Click on File->Export to HTML. Internet Explorer window pops up with HTML converted text. One needs to click on 'Export to HTML' each time when the conversion is required. Ctrl-H is the shortcut key. May also need to press the 'reload' button on the Internet Explorer to refresh. (Anybody knows a way to refresh explorer from the command line?)

Unicode Export

Click on File->Export to UTF8(Unicode). Internet Explorer window pops up with unicode test. One needs to click on 'Export to UTF8(Unicode)' each time when the conversion is required. Ctrl-U is the shortcut key. May also need to press the 'reload' button on the Internet Explorer to refresh. (Anybody knows a way to refresh explorer from the command line?). For more information: Chintha.com or Wikipedia.org

Formatting

Done by inserting HTML tags as comments. eg: To making a word bold, insert {<B>} before and {</B>} after that word. Remember {} avoids a word being converted to Malayalam and will be displayed in English. So the <B> tag will remain intact in the converted text and will display the word bold in Internet Explorer when 'Export to HTML' is clicked.

Printing

Convert to HTML. Then print using the Internet Explorer's print facility.

Editing a document already in Malayalam

Say you want to edit or search a document from www.malayalamanorama.com. For that, change font in Varamozhi Editor to "manorama". Copy paste the text from the site to Malayalam frame. Left click on the Malayalam frame once or twice. You get the correpsonding transliterated text in English frame. All the editing can be done in English frame or Malayalam frame alike.

Font Conversion

Convert the document to transliterated English text, as described above. Then change the font to the desired one - say Thoolika (font of Kaumudi publications). The document in Manorama is now in Thoolika font.

Search and replace

Right click on the English frame. Select the search option, enter the search string. Now select the Varamozhi Editor Window again. Matched text is highlighted.

How to install fonts?

Open Windows Start->Control Panel->Fonts. In the Fonts window, select File->Install New Font. Go to the directory containing fonts (eg: C:\Program Files\Varamozhi Editor\font). All the fonts in the directory will be recognised and listed. Select the desired and click OK. Done.

Where are the fonts?

Since the installation software corrupt already exisiting fonts during installation, I am not trying to install any font other than Chamundi. But all the fonts are supplied with the package. Goto font directory inside the installation directory.(eg: C:\Program Files\Varamozhi Editor\font)

Garbled text in Malayalam frame

This means some problem with the font installation. So check you have the font installed. If no, install it either from font's original site or from font directory inside the installation directory(eg: C:\Program Files\Varamozhi Editor\font). If not working still, uninstall font and reinstall it. If the problem is still not solved, reinstall Varamozhi Editor :(

Sending Malayalam documents

Export the document to HTML. Now there are different methods you could adopt:

  • Send the HTML document itself, if the person has the Malayalam font used.
  • Send the HTML document and the font used. Ask the receiving person to install the font before opening the document.
  • Open the HTML document in MSWord. Save As .doc document with Embed TrueType fonts checked in the 'Save As' options. Don't check the 'Embed only the characters used'. Send this MSWord document.
  • Install a printer-to-pdf redirect driver. PDF reDirect is an example. Then click File->Print the HTML document. Change the printer name to 'PDF reDirect'. This will save the document as a PDF file. Send this PDF file.
